# Google Sheets 协作指南

申请进度 Checklist 在 Excel 中生成，上传 Google Sheets 后与学生共享编辑。

## 上传方式

1. Google Drive → **新建 → 文件上传** → 选择 `{学生名}_申请进度Checklist.xlsx`
2. 双击打开 → **打开方式 → Google 表格**
3. 或：表格中 **文件 → 导入 → 上传** → 选择「替换电子表格」

建议保留 `.xlsx` 备份；Sheets 为协作副本。

## 共享权限 SOP

| 角色 | 权限 | 可编辑范围 |
|------|------|------------|
| 顾问 | Owner | 全部（公式列建议保护） |
| 学生 | 编辑者 | 网申状态、材料状态、账号、推荐信 Portal 状态 |
| 其他 | 评论者 / 无 | 按需 |

### 保护范围（推荐）

在 Sheets 中：**数据 → 受保护的工作表和数据区**

- 保护 `00_仪表盘` 全部（只读汇总）
- 保护 `01_项目总览` 的 Q–U 列（公式列：材料/LoR/Essay/就绪度）
- 保护各 Sheet 表头行

学生可编辑列示例：

- `01_项目总览`：K 网申状态、M 支付、N–O 提交日、V–Y 协作列
- `02_材料进度`：E 状态、G 本地路径、J 完成日期
- `05_网申账号`：F–J 列

## 导入后公式自检（必做 3 项）

上传完成后，逐项检查以下单元格 **不应** 显示 `#ERROR!` 或 `#REF!`：

### 1. 材料进度汇总

- **位置**：`01_项目总览` → **Q2**（第一个项目「材料进度」列）
- **预期**：显示 `0/8` 或类似 `已完成/总数` 格式
- **若报错**：检查 `02_材料进度` sheet 名是否为 `02_材料进度`（含中文）

### 2. 本周待办 — 学校列

- **位置**：`00_仪表盘` → 本周待办区 → **B 列**（第一个待办行的「学校」）
- **预期**：空白（距 DDL >14 天）或显示学校中文名
- **若报错**：多为 IF 公式缺 `)`；重新用 skill 脚本生成最新 xlsx

### 3. 仪表盘 DDL 倒计时

- **位置**：`00_仪表盘` → **J3**（距最近 DDL 天数）
- **预期**：数字（可为负表示已逾期）
- **若报错**：确认 `01_项目总览` G 列 Priority DDL 格式为 **日期**（非纯文本）

### 快速全表检查

选中 `01_项目总览!Q2:Q{末行}`，若均为 `#ERROR!`，重新下载 xlsx 并导入，勿在 Sheets 中手改公式。

## DDL 日期格式 fix

若 J3 报错而 G 列日期显示为文本：

1. 选中 `01_项目总览` G 列 → **格式 → 数字 → 日期**
2. 重新输入或从 xlsx Re-import

## 每周协作 SOP（15 分钟）

1. 打开 `00_仪表盘`，查看 J3 距最近 DDL
2. 滚动至「本周待办」，逐项处理临期项目
3. 学生更新：`05_网申账号`、材料/推荐信状态
4. 顾问更新：`04_Essay进度`、主表「当前卡点 / 下一步」
5. 更新 `最后更新人` 和 `最后更新日期`

## 定校变更后

1. 本地运行 `build_checklist.py --merge 当前Sheets导出的xlsx或本地备份`
2. 重新上传或仅替换有变化的 sheet（推荐整表 re-import 避免公式漂移）

## 安全提醒

- 不在表格中写明文密码
- 共享链接设「仅受邀用户」
- Application ID / 邮箱避免截图发公开群
